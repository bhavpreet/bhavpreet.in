# Bhavpreet Singh - Portfolio

Minimal artist portfolio website featuring film and photography work.

## Design

Japanese Zen aesthetic with maximum white space, subtle typography, and text-first navigation.

## Structure

- `/` - Homepage with text navigation
- `/films.html` - Film project index
- `/photography.html` - Photography series index
- `/about.html` - Artist bio
- `/films/` - Individual film project pages
- `/photography/` - Individual photography series pages

## Deployment

Hosted on GitHub Pages at [username].github.io/bhavpreet.in

## Local Development

Open `index.html` in a web browser. No build process required.
